for ordered list there are 2 attributes are there they are: type and start

type attributes are used for the list like starts with 1,2,3 or A,B,C or roman letters

start attributes are used to give a numeric value from which the list need to start for eg. start=5 means it starts from 5,6,7...

for A,B,C we wants to starts from letter e means we need to give a start value=5.

note: start value always in a numeric value only.

we can also reversed the list value by using a reversed keyword.
